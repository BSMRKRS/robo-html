Welcome to Robo-HTML

How to use:

Robo Readings:

- Run Readings.py in /robo-html/Readings/

Example: $ cd ~/robo-html/Readings && python Readings.py

View Camera:

- Run start_camera in home directory

Example: $ /home/student/start_camera.sh

Troubleshooting:

- Press ctrl + shift + r to hard refresh
